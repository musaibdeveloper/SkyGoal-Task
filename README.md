# SkyGoal-Task

## Steps

1. **Clone the Repo**

    ```bash
    git clone git@github.com:musaibdeveloper/SkyGoal-Task.git
    ```

2. **Install Dependencies**

    ```bash
    cd SkyGoal-Task
    cd server
    npm install
    ```

## Running Tests

To run tests, execute the following command:

```bash
npm start
```
